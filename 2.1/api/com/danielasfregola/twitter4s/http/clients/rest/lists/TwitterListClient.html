    Adds multiple members to a list.

    Adds multiple members to a list. The authenticated user must own the list to be able to add members to it. Note that lists can’t have more than 5,000 members, and you are limited to adding up to 100 members to a list at a time with this method. Please note that there can be issues with lists that rapidly remove and add memberships. Take care when using these methods such that you are not too rapidly switching between removals and adds on the same list.     list_id

    : The numerical id of the list.

    user_ids

    : The list of user IDs to add, up to 100 are allowed in a single request.
